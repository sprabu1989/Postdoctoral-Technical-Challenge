
# Models Folder

This folder contains all model implementations used in:

- Task 1: Vision Transformer classification
- Task 2: MedGemma medical report generation
- Task 3: CLIP and BiomedCLIP retrieval systems

Subfolders:

vision_transformer/
    ViT classification model and training

medgemma/
    Multimodal medical report generation utilities

clip_retrieval/
    CLIP / BiomedCLIP embedding and FAISS retrieval utilities
